Welcome to Readify

A simple and efficient document reader built with wxWidgets

About Readify

Readify is a lightweight document reader that allows you to view HTML documents with ease. It features a clean interface with navigation capabilities and is built using wxWidgets for cross-platform compatibility.

Features

  • Clean and intuitive user interface
  • HTML document viewing with wxHtmlWindow
  • File navigation tree
  • Toolbar with common actions
  • Status bar for file information
  • Cross-platform compatibility

Getting Started

You can open HTML files using the File menu or the toolbar. The navigation tree on the left shows available documents in the assets folder.
